Let's learn css colors

There are several ways to specify CSS colors:
1. CSS Color Names 2. RED Green and Blue 3. Hexadecimal (HEX) In CSS, a color can be specified as an R G B value, using this formula, using decimal numbering between 0 and 255 for each color.

RGB stands for: - Red - Green - Blue: The color black has a value of r g b(0,0,0) whereas white has a value of rgb(255,255,255).
